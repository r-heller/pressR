# List All Built-In Sensor Layouts

List All Built-In Sensor Layouts

## Usage

``` r
pr_layout_list()
```

## Value

A [tibble::tibble](https://tibble.tidyverse.org/reference/tibble.html)
with columns `name`, `system`, `grid_rows`, `grid_cols`, `n_sensors`,
`n_regions`, `description`.
