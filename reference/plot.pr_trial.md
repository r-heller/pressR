# Plot a Pressure Trial

Default plot method for `pr_trial` — shows the maximum pressure picture
(MPP) as a heatmap.

## Usage

``` r
# S3 method for class 'pr_trial'
plot(x, ...)
```

## Arguments

- x:

  A [pr_trial](https://r-heller.github.io/pressR/reference/pr_trial.md)
  object.

- ...:

  Passed to
  [`pr_plot_heatmap()`](https://r-heller.github.io/pressR/reference/pr_plot_heatmap.md).

## Value

A `ggplot2` object.
