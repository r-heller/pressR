# Total Contact Time

Duration for which at least one sensor exceeds `threshold`.

## Usage

``` r
pr_calc_contact_time(trial, threshold = 0)
```

## Arguments

- trial:

  A [pr_trial](https://r-heller.github.io/pressR/reference/pr_trial.md)
  object.

- threshold:

  Numeric. Default `0`.

## Value

Numeric scalar (s).
