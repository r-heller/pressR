# Plot Contact Area vs Time

Plot Contact Area vs Time

## Usage

``` r
pr_plot_contact_area(trial, show_cycles = FALSE)
```

## Arguments

- trial:

  A [pr_trial](https://r-heller.github.io/pressR/reference/pr_trial.md)
  object.

- show_cycles:

  Logical. If `TRUE`, shade detected stance phases. Default `FALSE`.

## Value

A `ggplot2` object.
